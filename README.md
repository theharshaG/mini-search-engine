# mini-search-engine

Mini Search Engine (Python Project)
## Overview

This project is a simple command-line Mini Search Engine implemented in Python. It allows users to store text data and search for specific keywords within the stored content. The data is saved in a text file to ensure persistence between program executions.

## Features

Add text data
Search data using keywords
Display matching results
Show total number of matches
Data persistence using file handling

## Technologies Used

Python
File Handling
String manipulation

## How to Run

install python (VS Code)
Run the program: mini search engine.py

## How It Works

User can add text data which is stored in a file (data.txt)
Each line in the file represents one entry

During search:
User enters a keyword
The program reads all data from the file
It checks each line to see if the keyword exists (case-insensitive)
Matching lines are displayed

If matches are found, total count is shown
If no match is found, a message is displayed
